# Screen Share Telegram Web App

Web приложение для демонстрации экрана через Telegram бота.

## Features
- 🖥️ Screen sharing
- 🪟 Window sharing
- 🔊 Audio transmission
- ⚡ Quality settings
- 📱 Responsive design

## Technologies
- HTML5, CSS3, JavaScript
- WebRTC API
- Telegram Web App API

## Setup
1. Clone repository
2. Configure GitHub Pages
3. Update Telegram bot URL
